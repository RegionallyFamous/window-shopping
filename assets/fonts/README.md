# Window Shopping Fonts

These font files are bundled so the theme does not depend on system fonts.

Sources:

- Archivo Black
- Barlow Condensed
- Cormorant Garamond
- Fraunces
- IBM Plex Mono
- IBM Plex Sans
- Inter
- Nunito Sans

The files were downloaded from Google Fonts as Latin `.woff2` subsets and are registered through `theme.json` `fontFace` declarations. These families are distributed under open font licenses by their respective authors. Keep this folder with the theme package so the Site Editor and front end render consistently.
